**What is the difference between csv.reader() and csv.DictReader()?**
csv.reader() reads each row of a file as a list. csv.DictReader() reads each row as a dictionary. It uses the first row of your file as the keys.
**Why are we using DictReader()?**
DictReader() to make reading data files much easier and clearer.
