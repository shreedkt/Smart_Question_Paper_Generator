**What is a Function?**
A function is block of code which is helpfull in reuse of the code block.
**What is a Module?**
Module is single file containg many functions, and other things like variable etc. 
**What is a Package?**
A Package is a folder which containing one or more module. It usually has __init__.py file
**What is a Project?**
A Project is a top level directory or folder where we build our software. It consist of all required module,package and configration related files.
